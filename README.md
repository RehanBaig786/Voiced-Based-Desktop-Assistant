# Voiced-Based-Desktop-Assistant<br>

Overview<br>

Elaris is an AI-powered voice assistant built using Python that combines speech recognition, natural language processing, automation, and real-time search to deliver a smart and interactive user experience.<br>
It is designed to simulate a human-like assistant capable of understanding and responding intelligently.<br>

Features<br>

-Speech Recognition (Voice → Text)<br>
-Text-to-Speech using Edge-TTS<br>
-Conversational AI (Groq / Cohere LLM)<br>
-Real-Time Internet Search<br>
-System & Application Automation<br>
-GUI Interface using PyQt5<br>
-Image Generation from prompts<br>
