Here is a simple Python program that checks if a given string is a palindrome:

```python
def is_palindrome(s):
    s = ''.join(c for c in s if c.isalnum()).lower()
    return s == s[::-1]

# Test the function
s = input("Enter a string: ")
if is_palindrome(s):
    print(s, "is a palindrome.")
else:
    print(s, "is not a palindrome.")
```

In this program, the `is_palindrome` function takes a string `s` as input and first removes all non-alphanumeric characters from the string and converts it to lowercase. It then checks if the resulting string is the same when reversed (`s[::-1]`). If it is, the function returns `True`, indicating that the string is a palindrome. Otherwise, it returns `False`.
